# Beauty Match - Back-end

Base inicial da API `beauty-match-back` seguindo o mesmo stack do projeto Profissa Livre.

## Stack
- NestJS 11
- TypeScript 5
- Jest para testes unitarios e e2e
- ESLint + Prettier
- Swagger

## Scripts principais
- `npm run start:dev`
- `npm run build`
- `npm run lint`
- `npm run test`

## Estrutura
- `src/`: codigo da aplicacao
- `test/`: testes e2e
- `.ai/`: regras e skills para IA
- `docs/`: documentacao de arquitetura e fluxo

## Primeiro commit
1. `cp .env.example .env`
2. `npm install`
3. `npm run lint && npm run test && npm run build`
